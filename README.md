# Atte
企業向けの勤怠管理アプリです。
<img width="1470" alt="Atte ホーム画面" src="https://github.com/user-attachments/assets/50d9424c-0df2-4983-bc59-bcc3823b1f2a">

## 作成した目的
企業における人事評価のため

## 機能一覧
・会員登録機能
</br>
・ログイン機能
</br>
・ログアウト機能
</br>
・勤務開始・終了時間の記録
</br>
・合計勤務時間の記録
</br>
・休憩開始・終了時間の記録
</br>
・合計休憩時間の記録
</br>
・日付別勤怠情報取得
</br>
従業員一覧表示
</br>
従業員別勤怠情報取得
